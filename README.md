# chardine.fr

[![Netlify Status](https://api.netlify.com/api/v1/badges/f3bba656-2789-495e-8bc3-3cc772c6d1dd/deploy-status)](https://app.netlify.com/projects/chardine/deploys)

My personal website and portfolio, built with [Next.js](https://nextjs.org), [Tailwind CSS](https://tailwindcss.com), and [next-intl](https://next-intl.dev). Available in 6 languages.

## Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS + shadcn/ui
- **i18n**: next-intl (en, fr, zh, es, de, kr)
- **Animations**: Framer Motion
- **Deployment**: Netlify

## Development

```bash
pnpm install
pnpm dev
```
